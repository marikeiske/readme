Objetivo

O objetivo do projeto é criar uma aplicação web para gerenciar tarefas pessoais.

Roadmap

O projeto será desenvolvido em duas etapas:

  • Criação da interface do usuário;
  • Implementação da funcionalidade de adicionar, editar e excluir tarefas.

Tecnologias

  • React
  • Redux
  • Node.js
  • MongoDB
